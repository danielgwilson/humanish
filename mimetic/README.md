# Mimetic Dogfood Config

This directory is the committed source plane for dogfooding `mimetic-cli` with
Mimetic itself.

Current scope:

- public-safe synthetic CLI personas;
- dry-run contract proof over the package, command tree, observer, and feedback
  issue path;
- no real Codex TUI actor yet;
- no OpenAI, E2B, GitHub mutation, provider spend, production data, or private
  artifacts.

Generated run bundles, observer HTML, review packets, logs, and local overrides
belong in ignored `.mimetic/`.

When live Codex TUI actor support exists, this config should become the first
4x persona harness target.
